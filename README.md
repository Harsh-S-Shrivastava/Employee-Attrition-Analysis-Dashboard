# Employee Attrition Analysis Dashboard

## Overview

This Power BI project focuses on analyzing employee attrition patterns within an organization. The dashboard provides insights into factors influencing employee turnover, helping HR teams identify high-risk groups and take data-driven retention actions.

## Objectives

* Analyze overall employee attrition trends
* Identify key factors contributing to turnover
* Examine attrition by department, role, gender, and tenure
* Support strategic HR planning and retention initiatives

## Tools Used

* Power BI Desktop
* Power Query
* DAX

## Data Source

The dataset consists of structured HR records (Excel/CSV) containing employee demographics, job roles, satisfaction levels, and attrition status.

## Future Scope

* Predictive modeling for attrition risk forecasting
* Integration with live HR systems
* Enhanced drill-through for employee-level insights
